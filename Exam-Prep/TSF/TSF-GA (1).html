from sklearn.metrics import mean_squared_error

# Split the data into train and test sets (using the last one-month data for testing)
test_start_date = df.index[-1] - pd.DateOffset(months=1)
train = df[df.index < test_start_date]
test = df[df.index >= test_start_date]

# Persistence model: Forecasting with the last observed value
train_mean = train['rainfall'].mean()  # Calculate the mean of the training data
predictions = np.full_like(test['rainfall'], fill_value=train_mean)  # Forecast using the mean

# Calculate residuals
residuals = test['rainfall'] - predictions

# Calculate RMSE (Root Mean Squared Error)
rmse = np.sqrt(mean_squared_error(test['rainfall'], predictions))

# Calculate MAPE (Mean Absolute Percentage Error)
def mean_absolute_percentage_error(y_true, y_pred): 
    return np.mean(np.abs((y_true - y_pred) / y_true)) * 100

mape = mean_absolute_percentage_error(test['rainfall'], predictions)

# Print RMSE and MAPE
print(f"RMSE: {rmse:.2f}")
print(f"MAPE: {mape:.2f}%")

# Plot residuals
plt.figure(figsize=(12, 6))
plt.plot(test.index, residuals, marker='o', linestyle='-')
plt.title('Residuals of Base Model (Persistence Model)')
plt.xlabel('Date')
plt.ylabel('Residuals')
plt.grid(True)
plt.show()

Interpretation and Comments: Model Explanation: Base Model (Persistence Model): This model forecasts the next value in the time series as the last observed value (train_mean in this case). Evaluation Metrics: RMSE (Root Mean Squared Error): RMSE measures the average magnitude of the residuals (prediction errors). Lower values indicate better fit. MAPE (Mean Absolute Percentage Error): MAPE measures the average magnitude of the percentage errors. Lower values indicate better accuracy. Residuals: The plot of residuals shows the differences between the predicted values and the actual values (test data). Ideally, residuals should be randomly scattered around zero without any clear pattern. Comments: RMSE and MAPE Values: Assess the RMSE and MAPE values in the context of the data and the specific domain. Lower values of RMSE and MAPE generally indicate better model performance, but this can vary depending on the nature of the data and the problem domain.

Persistence Model Performance: A persistence model serves as a baseline. It provides a simple benchmark for evaluating more complex models. The performance of the persistence model is typically moderate, and its adequacy depends on how well it captures the inertia or trend of the time series.

Further Steps: If the RMSE and MAPE are high, consider exploring more sophisticated models (e.g., ARIMA, exponential smoothing, or machine learning models) that can capture the seasonality, trends, and other complexities present in the data.
